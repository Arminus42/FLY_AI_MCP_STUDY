# FLY AI MCP STUDY

**FLY AI MCP 스터디** 공식 저장소입니다.  
이곳에서는 LLM과 외부 시스템을 연결하는 표준 프로토콜인 **MCP(Model Context Protocol)**를 학습하고, 실습한 코드를 공유합니다.

## 🛠 사전 준비
터미널에서 가상환경을 활성화하고 MCP 패키지를 설치합니다.
```bash
# 가상환경 생성 및 활성화 (선택)
python -m venv venv
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate

# 필수 패키지 설치
pip install mcp
```
